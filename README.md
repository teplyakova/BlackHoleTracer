# BlackHoleTracer
To build and execute BlackHoleTracer do following steps:
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    $ ./black_hole_tracer -c [PATH_TO_CONFIG] -o [OUTPUT_FILE] (for example ./black_hole_tracer -c ../data/beautiful.json -o output.png)
