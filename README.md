# BlackHoleTracer
To build and execute BlackHoleTracer do following steps in project folder:
    mkdir build
    cd build
    cmake ..
    make
    ./black_hole_tracer -c [config_file] -o [output_file] (for example ./black_hole_tracer -c ../data/beautiful.json -o output.png)
